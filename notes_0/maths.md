**eigenvector & eigenvalue**  
for square matrix `A`, eigenvector `u` & eigenvalue`λ`  
```
Au = λu

Au = λIu
Au - λIu = 0
(A - λI)u = 0
assuming u is non-zero, solve for λ using

det(Au - λI) = 0
```
